# Is the assembly loaded?

Is the assembly loaded?

## Usage

``` r
isAssemblyLoaded(assemblyName)
```

## Arguments

- assemblyName:

  The name of the assembly, e.g. 'ClrFacade'

## Value

TRUE if the assembly is loaded, FALSE otherwise

## Examples

``` r
isAssemblyLoaded("ClrFacade")
#> [1] TRUE
```
