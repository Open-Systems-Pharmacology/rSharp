# Gets the static properties for a type

Gets the static properties for a type

## Usage

``` r
getStaticProperties(objOrType, contains = "")
```

## Arguments

- objOrType:

  a \`NetObject\` object, or type name, possibly namespace and assembly
  qualified type name, e.g. 'My.Namespace.MyClass,MyAssemblyName'.

- contains:

  a string that the property names returned must contain
